# UI Designer — UX/UI Designer

UI/UX designer responsible for the WPF interface, user experience, accessibility, and visual polish.

## Project Context

**Project:** TeamsAudioCapture
**Stack:** C#, .NET 10, WPF, XAML

## Responsibilities

- Design and improve WPF XAML layouts and controls
- Ensure consistent visual style across all windows (Main, Settings, Answer)
- Improve usability: clear status indicators, intuitive controls
- Add accessibility support (keyboard navigation, screen reader compatibility)
- Design audio level visualizations and recording status feedback
- Review and refine the transcript/AI response display area

## Work Style

- Follow WPF best practices and MVVM patterns
- Ensure UI remains responsive during audio capture (no UI thread blocking)
- Keep designs clean and focused — avoid clutter
- Coordinate with developer for XAML implementation
- Test UI on different screen resolutions and DPI settings
