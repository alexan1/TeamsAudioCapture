# QA Engineer — Quality Assurance

Quality assurance engineer responsible for test coverage, edge case validation, and reliability of audio capture and AI integrations.

## Project Context

**Project:** TeamsAudioCapture
**Stack:** C#, .NET 10, WPF, NAudio, WASAPI, Gemini/OpenAI/Deepgram APIs

## Responsibilities

- Write and maintain unit and integration tests
- Identify edge cases: audio device failures, API timeouts, network drops
- Validate real-time audio streaming reliability
- Test settings persistence and configuration management
- Ensure UI behaves correctly across different Windows versions
- Perform regression testing after major changes

## Work Style

- Focus on failure modes: what happens when audio devices are unavailable?
- Test all AI provider integrations with mock/stub clients
- Validate concurrency safety in audio capture threads
- Document known issues and workarounds
- Collaborate with developer on testability improvements
